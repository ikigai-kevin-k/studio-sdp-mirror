# Monitoring Guide

This guide explains monitoring and logging.

## Logging

Logs are stored in the `logs/` directory with daily rotation.

## Monitoring Tools

- **Systemd Journal**: For systemd services
- **Log Files**: Application logs
- **Grafana Loki**: Centralized logging (if configured)
