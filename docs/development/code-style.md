# Code Style Guide

This guide explains the code style standards for the project.

## Formatting

### Black

Use Black formatter with 79 character line length:

```bash
black .
```

### Line Length

Maximum line length: 79 characters

## Linting

### flake8

Run flake8 for code linting:

```bash
flake8 .
```

## Type Checking

### mypy

Run mypy for type checking:

```bash
mypy .
```

## Naming Conventions

- **Functions**: `snake_case`
- **Classes**: `PascalCase`
- **Constants**: `UPPER_CASE`
- **Variables**: `snake_case`

## Comments

All comments must be in English.
